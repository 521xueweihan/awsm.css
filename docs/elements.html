Inlines

Of course the library has styles for strong and emphasized text.

Please note if you insert image inside p it'll be aligned to the left or right side. If it's in the start, then it's aligned to the left, else—to the start.

But if you insert image outside p it'll be aligned to the center. You may see example below.

There are many others stylized inline tags in lib, but it would be boring to list them all.
